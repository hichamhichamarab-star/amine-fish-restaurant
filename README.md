- ✅ تصميم احترافي وسريع
- ✅ حجز الطاولات أون لاين
- ✅ عرض القائمة الكاملة
- ✅ معلومات الاتصال والموقع
- ✅ تصميم متجاوب (Responsive)
- ✅ سهولة الاستخدام
- ✅ ديناميكي وآمن

## 📁 هيكل الملفات

```
amine-fish-restaurant/
├── index.html           # الصفحة الرئيسية
├── reservation.html     # صفحة الحجز
```

## 🚀 البدء السريع

### التثبيت المحلي

```bash
git clone https://github.com/hichamhichamarab-star/amine-fish-restaurant.git
cd amine-fish-restaurant
```

### الاستخدام

1. افتح ملف `index.html` في المتصفح
2. أو استخدم محرر نصوص مع Live Server

## 📝 الصفحات المتاحة

### 1. الصفحة الرئيسية (index.html)
- عرض جميل لمطعم
- الأطباق المميزة
- آراء العملاء
- الاشتراك في النشرة البريدية

### 2. صفحة الحجز (reservation.html)
- نموذج حجز الطاولات
- اختيار التاريخ والوقت
- معلومات حول ساعات العمل
- نصائح للعملاء

### 3. صفحة الاتصال (contact.html)
- نموذج التواصل
- معلومات الاتصال
- الموقع على الخريطة
- ساعات العمل
